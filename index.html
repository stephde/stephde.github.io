"Das Rotationsportal der Diplomaten"

"Das Rotationsportal" is a web app which supports diplomats with preparing their move to a new diplomatic mission. It provides a central location were information is gathered and structured. We developed the portal in a 3 month project which was a cooperation between Tech4Germany and the German Federal Ministry of Foreign Affairs.

Runtime Models in Microservice Systems

During my master`s thesis I dived into research regarding microservice sytems. For the thesis I developed a system that crawles data about a running microservices system from different sources (e.g. Kubernetes, Zipkin). From this data it automatically constructs a runtime model of the watched application. The model is exposed as a json and can be used to inspect the application or implement self-adaptation features.
